# 🌸 AIgul — AI-флорист, который понимает мужчин

Веб-приложение цветочного магазина с AI-агентом на базе **Google Gemini**.
Помогает мужчинам выбирать идеальные букеты для их женщин — задаёт уточняющие
вопросы, понимает контекст (ссора, годовщина, бюджет) и даёт **персонализированные**
рекомендации, а не просто кидает каталог.

---

## ✨ Что это решает

> Мужчины не умеют выбирать цветы. Они либо берут банальные красные розы,
> либо ходят 40 минут по магазину растерянные. Айгуль решает это за 2 минуты.

**Главный сценарий (демо для жюри):**
> *Клиент:* «Блин, мы жёстко поссорились, мне нужны цветы, чтобы извиниться, бюджет 10к тенге»
>
> *Айгуль:* не предлагает банальные красные розы. Понимает, что это извинения и нужен жест примирения, а не страсти. Рекомендует **белые пионы (символ примирения и чистых намерений)** — 7 шт за 9 500 ₸. Объясняет почему именно это, предлагает добавить открытку, помогает оформить.

## 📊 RAG из Excel — как магазин обновляет каталог

Магазин выгружает свой каталог в файл **`data/source/flowers.xlsx`** со следующими столбцами (русский или английский язык — алиасы поддерживаются):

| ID | Название (RU) | Название (KZ) | Цвет | Цена (тг) | Наличие | Повод | Кому дарить | Значение / Символика | Сезон |
|---|---|---|---|---|---|---|---|---|---|

При следующем запросе клиента приложение **автоматически перечитывает файл** (по mtime) и передаёт актуальный каталог в системный промпт Gemini. Никакого пересоздания, никакого ребилда — просто заменили `.xlsx` и магазин «переобучен».

В репо лежит готовый шаблон с 15 цветами. Регенерировать его можно командой:

```bash
node scripts/generate-flowers-xlsx.mjs
```

**Готовые букеты** (с реальными фото) лежат в `src/lib/bouquets.ts` — их собирают из позиций Excel-каталога. Когда AI рекомендует букет, на клиенте автоматически рендерится красивая карточка с фотографией.

## 🚀 Быстрый старт

### 1. Установка

```bash
npm install
```

### 2. Получите API-ключ Gemini

Бесплатный ключ: <https://aistudio.google.com/app/apikey>

### 3. Создайте `.env.local`

```bash
cp .env.example .env.local
```

И вставьте ключ:

```
GEMINI_API_KEY=your_api_key_here
```

### 4. Запустите dev-сервер

```bash
npm run dev
```

Откройте <http://localhost:3000>

## 🎯 Как работает AI-агент

1. **Системный промпт** (`src/lib/system-prompt.ts`) задаёт Gemini роль премиум-флориста-«вингмана»: эмпатия, юмор, вопросы по одному, никаких 10-вопросных анкет.
2. **RAG** (`src/lib/knowledge-base.ts`) — структурированный каталог магазина (цены, значения цветов, гайд по поводам, правила этикета) инжектится прямо в `systemInstruction`. В реальном проде здесь подключается векторная БД (Pinecone / pgvector), но для MVP-демо хватает.
3. **Чат** (`/api/chat`) — Next.js API Route шлёт историю в `gemini-2.0-flash-exp` через `startChat({ history })`.
4. **Передача контекста в checkout** — после рекомендации UI сохраняет историю чата в `localStorage`, страница оформления автоматически подтягивает её и подставляет в карточку «Рекомендация Айгуль».

Кнопки готовых сценариев уже встроены в чат:
